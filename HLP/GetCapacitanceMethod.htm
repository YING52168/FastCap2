FastCap2Object.GetCapacitance

VARIANT FastCap2Object.GetCapacitance()

Return Value

The method returns a 2-dimensional safe-array encapsulated in a variant data. The array contains the capacitance matrix as calculated by FastCap2, in Farads. The first dimension of the array is the row index, the second is the column index.

Notes:

The array dimensions can be queried from the safe array itself, see the 'safe array' topic in the MS Automation documentation.

Parameters

None

Remarks

Gets the capacitance values calculated by FastCap2, result of the simulation.
