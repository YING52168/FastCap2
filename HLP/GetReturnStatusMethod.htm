FastCap2Object.GetReturnStatus

short FastCap2Object.GetReturnStatus()

Return Value

The method returns the return status of FastCap2.

Valid status values are:

FC_NORMAL_END         0
FC_USER_BREAK        -1
FC_OUT_OF_MEMORY     -2
FC_CANNOT_OPEN_FILE  -4
FC_FILE_ERROR        -8
FC_GENERIC_ERROR     -8192
FC_STILL_RUNNING     -16384

Parameters

None

Remarks

Returns the reason why FastCap2 stopped execution. The reason could be normal end, user break or errors.
