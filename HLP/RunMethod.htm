FastCap2Object.Run

boolean FastCap2Object.Run(BSTR commandLine)

Return Value

True if FastCap2 was able to start a simulation, false otherwise.

Notes:

A reason why FastCap2 could not be able to start a simulation is that a simulation is already running. You can try to stop FastCap2 before retrying and / or poll FastCap2 status.

Parameters

commandLine

A string containing a FastCap2 command line, as would be given from a shell console.

Example:

FastCap2Object.Run("-lspheres.lst -o3")

Remarks

Runs a simulation.
