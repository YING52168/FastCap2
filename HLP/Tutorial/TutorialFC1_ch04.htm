Launch FastCap2 from FastModel

Launch FastCap2

Now you are ready to start a capacitance extraction. For this example, we will extract the capacitance of the sphere you have visualized in FastModel.

  1. Click on the icon on the toolbar. The Run Menu dialog from FastCap2 should open.

Select FastCap2 parameters

The FastCap2 Run Menu dialog contains a lot of options. For most simulations, you can keep the defaults. However, in our case you must inform FastCap2 that you are not feeding in a hierarchical input file but directly a geometry file.
  1. Remove the tick mark near the List File label, as shown in the figure.

Run FastCap2

  1. Click the Run button to start the simulation.
