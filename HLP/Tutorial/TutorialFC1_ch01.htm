FastCap2 Tutorial

Getting started with FastCap2

This tutorial will show you step-by-step how to perform the basic capacitance extraction tasks with FastCap2.

In this tutorial, you will learn how to:

  • Load a FastCap2 input file into FastModel
  • Understand basic FastCap2 input file syntax
  • aunch FastCap2 from FastModel
  • Understand FastCap2 output
  • Refine the geometry to achieve greater accuracy
  • Change the global permittivity factor
  • Work with hierarchical input files (list files)
  • Define dielectric interfaces
