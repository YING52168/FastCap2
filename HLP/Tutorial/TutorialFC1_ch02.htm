Load a FastCap2 input file into FastModel

Launch FastModel

  1. If FastModel is not already running, double click the FastModel icon on your desktop or select 'FastModel' from your Start menu in Windows.
  2. Click on the I agree button to accept FastModel's license.

Load a FastCap2 input file

  1. Select the FastCap2 View (the view should be in foreground with the caption highlighted).
  2. From the File menu select File->Open FastCap2 File or click on the icon on the toolbar.
  3. Under the Files of type drop-down menu select FastCap2 Geometry Files (*.neu, *.qui)
  4. Browse to the software installation directory (e.g. C:\Program Files\FastFieldSolvers) and under FastCap2\Samples\Input_files select the file sphere1.qui, then click Open.

    A coarse model of a sphere will appear in the FastModel View.
