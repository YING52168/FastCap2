Change the global permittivity factor

We have calculated the capacitance of a sphere in free air. Now we would like to see how this capacitance would change if the sphere was embedded in a different dielectric medium, whose relative dielectric constant is different from one.

  1. With the file named sphere3.qui open in FastModel, launch FastCap2 but stop at the Run Menu dialog.
  2. Change the global permittivity factor to 2, as indicated in the following figure.
  3. Launch the simulation by clicking on the Run button.
  4. Examine the result. The calculated capacitance is 0.2218 nF, exactly twice the 0.1109 nF value we obtained in the free air, as expected.
